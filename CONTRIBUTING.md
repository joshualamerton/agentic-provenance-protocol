# Contributing to APP

Thank you for your interest in the Agentic Provenance Protocol.

APP is an open protocol and community-driven initiative focused on establishing trust infrastructure for autonomous systems.

## Ways to Contribute

We welcome contributions in the following areas:

### Protocol Design

* RFC proposals
* Protocol improvements
* Security reviews
* Federation models
* Governance frameworks

### Reference Implementations

* Node implementations
* Storage backends
* Federation services
* SDK improvements

### Documentation

* Specifications
* Tutorials
* Examples
* Architecture diagrams

### Security

* Vulnerability reviews
* Cryptographic audits
* Threat modeling
* Protocol analysis

## RFC Process

Major protocol changes should begin as RFC proposals.

RFCs should include:

* Motivation
* Specification
* Security considerations
* Compatibility considerations

## Pull Requests

Before submitting significant changes:

1. Open an Issue
2. Discuss the proposal
3. Reach consensus where possible
4. Submit a Pull Request

## Code Standards

* Clear and maintainable code
* Comprehensive documentation
* Security-first design
* Backward compatibility where practical

## Code of Conduct

Contributors are expected to engage respectfully and constructively.
