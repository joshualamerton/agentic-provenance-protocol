# Agentic Provenance Protocol (APP)

The Trust Layer for Autonomous Systems.

APP is an open protocol for agent identity, ownership, provenance, authorization, reputation, delegation, and federation.

Just as SSL secured websites and OAuth secured applications, APP secures autonomous systems.

## Core Components

- Agent Passports
- Provenance Ledger
- Ownership Verification
- Federation
- Attestations
- Reputation
- Delegation
- Spend Authorization

## Mission

To create open standards for trust, identity, ownership, and provenance in the emerging agent internet.

## Status

APP is currently under active development.

The specification, reference node, SDKs, and governance framework will be published incrementally.
